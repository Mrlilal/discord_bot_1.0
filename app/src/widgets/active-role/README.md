##ACTIVE-ROLE

	# Setup

	```js
	[
	  {
		"guildId": "GUILD_ID",
		"roleId": "ROLE_ID",
		"roleDuration": 43200
	  },
	  // ...Add as many rules as you want.
	]
	```
	
	- `roleDuration` (in seconds) is how long the role is given to the member before it is removed.
	- `npm start` to run the bot.
